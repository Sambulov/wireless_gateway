# ESP32 Web Terminal

Веб-интерфейс терминала для ESP32 с поддержкой UART через WebSocket.

## Функции
- Подключение к WebSocket серверу ESP32
- Автоматическая регистрация клиента
- Отправка и получение данных в формате Base64
- Поддержка специальных клавиш (Ctrl+C, Ctrl+L)
- Подсветка синтаксиса и цветовая схема терминала
- Статистика соединения

## Настройка
1. Убедитесь, что ESP32 сервер доступен по WebSocket
2. Измените URL в поле ввода (по умолчанию: `ws://192.168.4.1/ws`)
3. Нажмите "Подключиться"

## API протокола
- Регистрация клиента: `{"FID":"0x00001021"}`
- Отправка данных в UART: `{"FID":"1022","ARG":"base64_data"}`
- Получение данных из UART: `{"FID":"0x00001021","SID":"...","ARG":"base64_data"}`

## Горячие клавиши
- `Ctrl+C` - Отправить сигнал Break
- `Ctrl+L` - Очистить экран терминала
- `Ctrl+U` - Очистить текущую строку

## Запуск
Просто откройте `index.html` в современном браузере.
